# GST Platform Constitution

## Constitutional Rules

### Rule 1

Everything must be registry driven.

### Rule 2

No business source may be hardcoded.

### Rule 3

No provider may be hardcoded.

### Rule 4

All external integrations must use adapters.

### Rule 5

Every data change must be versioned.

### Rule 6

Every document must be preserved.

### Rule 7

Every notification must be traceable.

### Rule 8

Every GST answer must be explainable.

### Rule 9

Every module must be independently deployable.

### Rule 10

Failure of one provider must not stop platform operation.

## Governance

Platform behavior is governed through:

- Registry
- Workflow Definitions
- Source Definitions
- Provider Definitions
- Schedule Definitions

Never through code changes.
