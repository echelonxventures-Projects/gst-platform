Platform Services

These are the services available via API to integrated clients.

HSN Code Lookup

Get GST/tax rate for any HSN (Harmonised System of Nomenclature) code. Returns CGST, SGST, IGST rates.

GET /api/v1/hsn/:code   GET /api/v1/hsn?rate=18

SAC Code Lookup

Get GST rate for SAC (Services Accounting Code) for service-based businesses.

GET /api/v1/search?q=consulting

Product/Service Search

Search HSN/SAC by product or service name. Supports auto-suggest for typeahead.

GET /api/v1/search?q=laptop   GET /api/v1/suggest?q=mob

State-wise GST Rules

Get state-specific GST rules, exemptions, and special rates by state code.

GET /api/v1/states/:stateCode/rules

GST Notifications

Latest CBIC GST notifications, circulars, and rate changes.

GET /api/v1/notifications

Integration Guide

Share this with client developers to integrate GST/tax services into their website.

1. Authentication

All requests need the X-Api-Key header with the key generated in Customers section.

curl -H "X-Api-Key: gst_your_api_key_here" \
  https://your-domain.com/api/v1/hsn/8471

2. Available Endpoints

ServiceEndpointDescription
HSN LookupGET /api/v1/hsn/:codeGST rate by HSN code
HSN by RateGET /api/v1/hsn?rate=18All codes at given rate
SearchGET /api/v1/search?q=laptopSearch by product/service name
SuggestGET /api/v1/suggest?q=mobTypeahead suggestions
State RulesGET /api/v1/states/:code/rulesState-specific rules
NotificationsGET /api/v1/notificationsLatest GST notifications

3. Response Example

// GET /api/v1/hsn/8471
{
  "hsn": "8471",
  "description": "Automatic data processing machines",
  "gst_rate": 18,
  "cgst": 9, "sgst": 9, "igst": 18,
  "effective_from": "2017-07-01"
}

4. JavaScript Example

const res = await fetch('https://your-domain.com/api/v1/hsn/8471', {
  headers: { 'X-Api-Key': 'gst_your_key' }
});
const data = await res.json();
const gstRate = data.gst_rate; // 18

5. Rate Limit Headers

Every response includes X-RateLimit-Limit and X-RateLimit-Remaining.
